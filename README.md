🍔 Foodie - React Food Ordering App

Foodie is a responsive food ordering web application built using React.js. The application allows users to browse delicious food items, add them to the cart, update quantities, and place orders through a simple checkout process.



- 🏠 Responsive Home Page
- 🍽️ Browse Food Menu
- 🛒 Add, Update, and Remove Cart Items
- 💰 Automatic Total Price Calculation
- 📦 Checkout Form with Validation
- ⚡ React Router Navigation
- 🚀 Performance Optimization using Lazy Loading, Suspense, React.memo(), useMemo(), and useCallback()
- 📱 Responsive Design



- React.js
- React Router DOM
- CSS / Bootstrap
